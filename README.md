Das Projekt ist eine Tresor-Applikation zur sicheren Speicherung sensibler Daten wie Passwörter, 
Zugangsdaten und Notizen. Benutzer authentifizieren sich mit E-Mail und Passwort, 
anschliessend werden Requests über JWT abgesichert und rollenbasiert autorisiert. 
Sicherheitskritische Funktionen wie Passwort-Hashing und Verschlüsselung der Secrets werden ausschliesslich im Backend umgesetzt. 
Ziel des Projekts ist es, auf einer bestehenden Basisapplikation schrittweise zentrale Security-Mechanismen zu implementieren und nachvollziehbar zu dokumentieren.
